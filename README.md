## 💫 About Me

> *I have always believed that the best way to learn is to build.*

That mindset has taken me from writing my first programs to developing full-stack web applications, exploring data science, and now diving deep into artificial intelligence and machine learning. Every project I complete teaches me something new, and every bug is simply another puzzle waiting to be solved.

As a **Data Science student**, I'm passionate about transforming raw data into meaningful insights and intelligent solutions. I enjoy working across the complete development lifecycle — from understanding a problem and designing a solution, to building, testing, optimizing, and continuously improving it.

Whether I'm building responsive web applications, analyzing datasets with Python and SQL, or training neural networks, I always want to understand **not only *how* something works, but *why* it works.**

My recent focus has shifted toward **Deep Learning and Computer Vision**, where I've been building projects using **PyTorch** and exploring how modern AI systems learn from data. Beyond that, I'm fascinated by **Large Language Models (LLMs)** and the future of generative AI — not just to train models, but to build intelligent systems that solve meaningful real-world problems.

I'm the type of person who can spend hours debugging a single issue, only to discover it was one misplaced character — and *somehow still consider it a productive day* because I learned something along the way.

> **"The best code isn't just the one that runs — it's the one that teaches you something every time you write it."**

## <a id="deep-learning"></a>🧠 Deep Learning & Neural Networks

<details open>
<summary><b>🔷 Neural Network from Scratch</b></summary>
<br>

A foundational deep learning project implementing a feedforward Neural Network (NN) from scratch — forward propagation, backpropagation, gradient descent, and activation functions — built to understand exactly how deep learning frameworks work under the hood.

`Python` `NumPy` `Matplotlib`

</details>

<details>
<summary><b>🔶 Binary Image Classifier</b></summary>
<br>

A binary classification project using deep learning to distinguish between two image classes. Covers data augmentation, CNN architecture design, dropout/regularization, and evaluation via accuracy, precision, recall, and ROC-AUC.

`Python` `TensorFlow` `Keras` `CNN` `OpenCV`

</details>

<details>
<summary><b>🔷 Multiclass Image Classification with CNN</b></summary>
<br>

A Convolutional Neural Network built to classify images across multiple categories — convolution & pooling layers, transfer learning experiments, softmax output, confusion matrix analysis, and hyperparameter tuning.

`Python` `TensorFlow` `Keras` `CNN` `Scikit-learn`

</details>
